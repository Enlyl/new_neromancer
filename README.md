## О проекте

`DS://CORP` — это одностраничный интерактивный курс по Data Science, оформленный как RPG-игра.
Вместо скучных лекций — модули, XP, уровни, достижения, магазин и ежедневные квесты.

Проект создан как личный эксперимент: что если превратить процесс обучения в игру?

---

## Механики

| Система | Описание |
|---|---|
| **Модули** | 10 тем DS, от основ до продвинутых |
| **XP & уровни** | Прогресс за прочитанные темы и тесты |
| **Достижения** | Разблокируются по мере прохождения |
| **Магазин** | Аватары, титулы, эффекты интерфейса |
| **Темы** | Cyber, Matrix, Red Team, Deep Sea, Ghost, Retro |
| **Квест дня** | Ежедневные задания с бонусным XP |
| **Глоссарий** | Справочник терминов DS |
| **Шпаргалка** | Быстрый доступ к нужным командам |
